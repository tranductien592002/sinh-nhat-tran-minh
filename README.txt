3) Drag the unzipped folder into https://app.netlify.com/drop or upload files into a GitHub repo root and enable GitHub Pages (branch main, /root).
4) Open the public URL returned by Netlify or GitHub Pages and test. If audio doesn't autoplay, click "Mở thiệp & Bật nhạc" or "Play music".
